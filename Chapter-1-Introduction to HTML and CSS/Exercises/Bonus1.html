HTML & CSS Guide


HTML (the Hypertext Markup Language) and CSS (Cascading Style Sheets) are two of the core technologies for building Web pages. HTML provides the structure of the page, CSS the (visual and aural) layout, for a variety of devices. Along with graphics and scripting, HTML and CSS are the basis of building Web pages and Web Applications.

Introduction to HTML

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser. It is frequently assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript. Here are various uses of HTML.



HTML Tags Quick Guide


1. < p >: This tag is used to write paragraphs.
For Example: 


                <body>
                    <p>This is a paragraph.</p>
                </body>

2. < ul >: This tag is used for creating an unordered list.
For Example: 

                <body>
                    <ul>
                        <li>One</li>
                        <li>Two</li>
                    </ul>    
                </body>

3. < ol >: This tag is used for creating an ordered list.
For Example: 

                <body>
                    <ol>
                        <li>Three</li>
                        <li>Four</li>
                    </ol>    
                </body>

4. < li >: This tag defines a list item inside an ordered or unordered list.
For Example: 

                <body>
                    <ol>
                        <li>Three</li>
                        <li>Four</li>
                    </ol>    
                </body>

5. < b >: This tag is used to make the text bold.
For example: 

                <p><b>This is a bold text.</b></p>

6. < i >: This tag is used to make the text italic.
For Example: 

                <i>This text is in italics.</i>

7. < !-- ... -- >: This tag is used to add comments in the code. The comments are not visible in the output. 

                <!--This is a comment.-->

8. < h1 > to < h6 >: This tag defines HTML headings. h1 is the largest heading and h6 is the smallest.
For Example: 

                <body>
                    <h1>Heading.</h1>
                </body>

9. < img >: This tag defines an image. It is used to an image to a webpage.
For Example: 

                <body>
                    <img src="link">
                </body>

10. < link >: This tag defines the relatioship between a document and an external resource. It is mostly used to link to style sheets.
For Example:

                <link href="link.css">

12. < a >: This tag defines a hyperlink.
For Example: 

                <a href="link">Text</a>



Introduction to CSS

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML. CSS is a rule-based language â€” you define the rules by specifying groups of styles that should be applied to particular elements or groups of elements on your web page.



CSS Properties Quick Guide


1. background-color: This property specifies the background color of an element. For Example:
body {background-color: #000000;}

2. background-image: This property specifies one or more background images for an element. For Example:
body {background-image: url("link");}

3. border: This is a shorthand property for border-width, border-style and border-color. For Example:
h1 {border: 2px solid blue;}

4. border-radius: This is a shorthand property for the four border-radius properties. For Example:
h1 {border: 2px solid blue;
border-radius: 10%; }

5. color: This property sets the color of the text.
For Example:
p {color: green;}

6. font-family: This specifies the font family for the text.
For Example:
p {font-family: Arial;}

7. font-size: This property specifies the size of the text.
For Example:
p {font-size: 20px;}

8. font-weight: This property is used to specify the weight of the text.
For Example:
p {font-weight: bold;}

9. height: This property helps in setting the height of the element.
For Example:
p {height: 400px;}

10. width: This property helps in setting the width of the element.
For Example:
p {width: 300px;}

11. list-style: This sets all the properties for a list in one declaration.
For Example:
ul {list style: none;}

12. margin: This sets all the margin properties in one declaration.
For Example:
h1 {margin: auto;}
